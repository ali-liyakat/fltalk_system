FLTalk

for reproducible federated experiments

A a lightweight, GUI based Federated Learning framework designed for running FL experiments locally with multiple clients.

  • Local server and flexible number of clients
  • Configurable models, algorithms, and datasets
  • Simple experiment execution and monitoring
Federated topology Server ↔ Clients

Workflow

How FLTalk works

A simple, repeatable workflow for running federated experiments locally.

01

Download the code

Get the server and a single client template from GitHub.

02

Duplicate clients

Copy the client directory to create as many clients as you want.

03

Configure experiment

Set model, algorithm, dataset, and the server URL inside Client and Server files.

04

Run & monitor logs

Run server and clients locally, then view logs on the web page.

Configuration

Configure your experiment

Select a model, aggregation algorithm, and dataset. Then adjust parameters inside the code.
