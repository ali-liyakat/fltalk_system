# FLTalk

FLTalk is a web-based federated learning framework that enables users to run federated learning experiments using locally configured clients and an aggregation server.

## Features

- FastAPI-based main server
- Web-based user interface
- Model and aggregation algorithm repository
- Client and server monitoring
- Experiment logs and round-wise metrics
- Local training with private datasets

## System Workflow

1. Open the FLTalk web interface.
2. Download the federated client and server templates from GitHub.
3. Configure the backend URL, dataset, model, and training parameters.
4. Run the federated server locally.
5. Run one or more federated clients locally.
6. Monitor experiment status, logs, and metrics through the FLTalk GUI.
